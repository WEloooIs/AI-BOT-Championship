Snapshot of the `original` local bot project.

Included:
- original `pyla_main.exe`
- `_internal` runtime payload
- bot configs, models, and support scripts
- surface launcher patches in `original_runtime_patch.py`

Excluded from git:
- `pyla_main.exe_extracted/`
- runtime logs
- `runtime_state/`

Start points:
- `start_original.bat` for the untouched original executable path
- `start.bat` for the patched launcher path
- `hub.bat` for the local settings UI
